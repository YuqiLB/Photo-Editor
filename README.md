# Photo Editor

A web-based photo editor with image upload and editing capabilities.

## Project Structure

```
Photo-Editor/
├── vite-project/    # React frontend
├── backend/         # Express backend server
└── web/            # (clarify purpose)
```

## Setup

### Frontend
```bash
cd vite-project
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Tech Stack

- **Frontend**: React, Vite
- **Backend**: Node.js, Express, Multer
- **Styling**: CSS

## Features

- Drag & drop image upload
- Multiple image selection
- Image preview
- Image deletion
